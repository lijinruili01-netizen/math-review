# 高等数学C复习平台 - 分享指南

## 📦 分享给朋友的文件清单

请将以下文件打包发送给朋友：

```
数学复习/
├── index.html           ✅ 必需
├── style.css            ✅ 必需
├── script.js            ✅ 必需
├── server.py            ✅ 必需
├── requirements.txt     ✅ 必需
├── 启动服务器.bat       ✅ 必需（Windows用户）
└── README.md            ✅ 必需（使用说明）
```

## 🎁 两种分享方案

### 方案一：让朋友使用自己的API密钥（推荐⭐）

**优点**：
- 不消耗你的API配额
- 更安全
- 朋友可以独立使用

**步骤**：

1. **修改 `server.py`**，让API密钥可配置：
   - 我已为你准备了新版本的 `server.py`（见下方）
   
2. **创建 `config.py`** 文件模板供朋友填写

3. **发送文件** + **README.md** 说明

### 方案二：直接分享（使用你的API密钥）

**优点**：
- 朋友使用简单，无需申请API

**缺点**：
- ⚠️ 消耗你的DeepSeek API配额
- ⚠️ 如果朋友大量使用，可能产生费用
- ⚠️ API密钥泄露风险

**步骤**：
1. 直接打包所有文件
2. 发送给朋友 + README.md

## 📝 创建分享包的步骤

1. **创建压缩包**：
   - 右键点击"数学复习"文件夹
   - 选择"发送到" → "压缩(zipped)文件夹"
   - 生成 `数学复习.zip`

2. **发送文件**：
   - 通过微信/QQ/邮件发送压缩包
   - 一并发送 `README.md`（使用说明）

3. **告知朋友**：
   - 需要安装Python 3.7+
   - 解压后双击 `启动服务器.bat`
   - 在浏览器访问 http://localhost:8000

## 🔧 朋友的使用要求

### 必需环境
- ✅ Windows 系统
- ✅ Python 3.7 或更高版本
- ✅ 网络连接（用于调用AI）

### 安装Python
如果朋友没有Python：
1. 访问 https://www.python.org/downloads/
2. 下载并安装Python（记得勾选"Add Python to PATH"）
3. 重启电脑

## 📋 朋友的使用步骤

1. **解压文件**到任意文件夹
2. **双击运行** `启动服务器.bat`
3. **等待**服务器启动（约5秒）
4. **打开浏览器**访问 http://localhost:8000
5. **开始使用**！

## ⚠️ 重要提醒

### 如果使用方案一（朋友自己的API密钥）
- 朋友需要先注册DeepSeek账号：https://platform.deepseek.com/
- 获取API密钥后填写到 `config.py`
- 费用由朋友自己承担

### 如果使用方案二（你的API密钥）
- ⚠️ 监控你的API使用量
- ⚠️ 设置使用限制
- ⚠️ 定期检查账单

## 🆘 常见问题

**Q: 双击bat文件闪退？**
A: 可能是Python未安装或未添加到PATH

**Q: 显示端口被占用？**
A: 关闭其他占用8000或5000端口的程序

**Q: AI不回复？**
A: 检查网络连接，确认API密钥有效
