## ⚙️ 配置说明（可选）

### 使用自己的API密钥

如果你想使用自己的DeepSeek API：

1. **注册账号**：访问 https://platform.deepseek.com/
2. **获取API密钥**
3. **编辑 `server.py`**，修改第8行：
   ```python
   'api_key': '你的API密钥',
   ```
4. **重启服务器**

### 修改端口

如果端口被占用，可以修改 `server.py` 最后一行：
```python
app.run(host='0.0.0.0', port=5000, debug=True)  # 改为其他端口
```

也要修改 `script.js` 第3行：
```javascript
endpoint: 'http://localhost:5000/api/chat',  // 改为对应端口
```

## 🔧 故障排除

### 问题：双击bat文件闪退

**原因**：Python未安装或未添加到PATH

**解决方法**：
1. 下载安装Python：https://www.python.org/downloads/
2. 安装时勾选"Add Python to PATH"
3. 重启电脑

### 问题：端口被占用

**错误信息**：`Address already in use`

**解决方法**：
```powershell
# 查找占用端口的进程
netstat -ano | findstr :8000
netstat -ano | findstr :5000

# 结束进程（替换<PID>为实际进程ID）
taskkill /PID <PID> /F
```

### 问题：AI显示"Failed to fetch"

**可能原因**：
- 后端服务器未运行
- 网络连接问题
- API密钥无效

**解决方法**：
1. 确认命令行窗口显示两个服务器都在运行
2. 检查网络连接
3. 验证API密钥是否有效

### 问题：依赖安装失败

**解决方法**：
```powershell
# 手动安装依赖
pip install flask flask-cors requests
```
